**Code Verification**:
```dart

class FirestoreHelper {
  static CollectionReference<Map<String, dynamic>> _usersCollection() {
    return _db.collection('users');
  }
  
  static DocumentReference<Map<String, dynamic>> userDoc(String userId) {
    return _usersCollection().doc(userId);
  }
}
```

### 3. Singleton Service Pattern (Similar to React Context) ✅

**Status**: COMPLIANT

**Implementation**:
- ✅ `AuthService` uses singleton pattern (equivalent to React Context)
- ✅ `FirebaseService` uses singleton pattern
- ✅ Centralized state management
- ✅ Single source of truth for auth state

**Code Verification**:
```dart
// ✅ CORRECT - Singleton pattern
class AuthService {
  static AuthService? _instance;
  static AuthService get instance {
    _instance ??= AuthService._();
    return _instance!;
  }
  
  AuthService._(); // Private constructor
}

// Usage (similar to React Context)
AuthService.instance.signInWithEmailAndPassword(...);
AuthService.instance.currentUser; // Single source of truth
```

**Files Checked**:
